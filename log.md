# 系統發展紀錄 (Log)

此檔案用於記錄系統版本更迭及已完成的發展工作。

## 版本紀錄

- **v1.0.0** (單機版建置)
  - 系統基礎架構建置完成。
  - Docker 環境與腳本 (Windows bat) 整合完畢。

## 已完成發展紀錄

- [x] **後端 (FastAPI)**
  - RESTful API 端點建置 (商品、原料、訂單、進貨、發票、報表)。
  - SQLAlchemy 資料庫模型與 Alembic Migration。
  - 庫存進銷存邏輯 (BOM 扣減、WAC 成本計算)。
  - 財務報表生成 (毛利分析、庫存估值、401 申報匯出)。
- [x] **前端 (React + Vite)**
  - 收銀介面實作 (購物車、結帳操作)。
  - 後台管理介面 (商品管理、庫存管理、財務報表)。
  - 與後端 API 的串接設定。
- [x] **單機版封裝 (Windows)**
  - 提供 `setup.bat` 與 `start.bat` 一鍵安裝啟動機制。
  - 支援自動化設定 Desktop 捷徑 (`create_shortcut.ps1`)。
  - 提供資料庫每日自動備份腳本及還原說明。
